==== 用户上传体温信息方法
....
GET /user/login/addtemperature
....


===== Description

===== Parameters

[options="header", cols=".^2,.^3,.^9,.^4"]
|===
|Type|Name|Description|Schema
|**Query**|**userid** +
__optional__|绑定用户id，服务器判断|integer (int32)
|**Query**|**上传时间** +
__optional__|信息的上传时间，由服务器判断|string
|**Query**|**地点** +
__required__|疫情上传温度实体类的地点属性|string
|**Query**|**温度** +
__required__|疫情上传温度实体类的温度属性|string
|===

|**404**|Not Found|No Content
|===
